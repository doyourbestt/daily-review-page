# 今日复盘模块重构 - 富兰克林美德系统

## 一、概述

将原有的开放问答式复盘改为**结构化评分式复盘**，参考富兰克林13条美德，结合现代职场个人成长需求，建立一套可量化、可追踪的美德实践体系。

## 二、核心设计

### 2.1 交互方式
- **随机抽选**: 每天随机抽取 1-3 个领域进行复盘
- **评分标准**: 每题 1-5 分（1=完全没做到，5=完美执行）
- **输出形式**: 混合模式（简洁总结 + 可展开详情）

### 2.2 领域定义（10大领域）

| 领域 | 英文名 | 核心价值 | 题目数量 |
|------|--------|----------|----------|
| 运动健身 | Fitness | 身体是革命的本钱 | 5题 |
| 阅读学习 | Learning | 持续成长输入 | 5题 |
| 人际沟通 | Relationships | 关系质量维护 | 5题 |
| 工作效能 | Productivity | 职业价值创造 | 5题 |
| 睡眠休息 | Sleep | 精力恢复管理 | 5题 |
| 目标执行 | Goals | 计划落地能力 | 5题 |
| 冥想正念 | Mindfulness | 内心平静专注 | 5题 |
| 习惯养成 | Habits | 微小积累坚持 | 5题 |
| 创意输出 | Creativity | 价值创造表达 | 5题 |
| 协作共赢 | Collaboration | 团队合作能力 | 5题 |

## 三、题库详情

### 3.1 运动健身 (Fitness)

**目标**: 身体是革命的本钱

| # | 问题 | 评分标准 |
|---|------|----------|
| 1 | 今天是否进行了运动？ | 1=没运动，2=拉伸/散步，3=运动<30分钟，4=30-60分钟，5=60分钟以上 |
| 2 | 运动强度如何？ | 1=无，2=轻度（散步），3=中等（慢跑/瑜伽），4=较高（健身/球类），5=高强度（HIIT/长跑） |
| 3 | 运动后身体感受？ | 1=疲劳加重，2=无感，3=轻微舒适，4=明显舒适，5=神清气爽 |
| 4 | 是否规律运动（每周3次以上）？ | 1=放弃状态，2=偶尔一次，3=偶尔规律，4=基本规律，5=非常规律 |
| 5 | 是否有意识地保持正确姿势？ | 1=完全忽视，2=偶尔注意，3=有时注意，4=经常注意，5=时刻保持 |

**实践标准（输出详情）**:
- 优秀: 运动≥30分钟，强度适中，体感舒适，规律坚持
- 良好: 有运动习惯，保持正确姿势
- 待改进: 运动不足，需要增加频次或时长
